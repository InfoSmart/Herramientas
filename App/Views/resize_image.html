Redimensionar una imagen

Esta herramienta te permite redimensionar una imagen haciendo una consulta hacia una dirección web local que debe contener la dirección web de la imagen a redimensionar y las dimensiones nuevas.

Es decir:

%PATH%/to/resize_image?url=dirección web&resize=ancho x altura

Por ejemplo, si quisieramos redimensionar el Logo de InfoSmart que esta ubicado en http://resources.infosmart.mx/infosmart/images/logo.png a las dimensiones 100x50 la dirección debería ser escrita de esta manera:

%PATH%/to/resize_image?url=http://resources.infosmart.mx/infosmart/images/logo.png&resize=100x50

Ahora para usarla en tu aplicación web puedes hacerlo mediante una etiqueta de imagen:
