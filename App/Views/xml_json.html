Transformando XML a JSON.

Funcionamiento

Esta herramienta te permite transformar un archivo/dirección XML a formato JSON haciendo una simple petición al servicio almacenado localmente.

La ventaja de esta herramienta es que puedes obtener de forma sencilla y rápida el contenido JSON de un archivo o dirección web en XML sin modulos pesados ni edición de código en tu servidor.

Ideas de uso

Esta herramienta puede ser perfecta para obtener Feeds de noticias en XML y transformarlas a JSON para su fácil manipulación en lenguajes como PHP o JavaScript.

InfoSmart usa esta tecnología en el Slide de noticias en la página principal.

Parámetros

Dirección web base a solicitar:

%PATH%/to/xml_json

A partir de la dirección web (mostrada justo arriba) puedes agregar los siguientes parámetros:

  • url Dirección web del sitio XML. (Obligatorio)
  • entities Si su valor se encuentra en true el contenido del JSON será codificado en HTML ENTITIES. Predeterminado: true
  • attr Si su valor se encuentra en true devolverá los atributos originales del XML. Predeterminado: false
  • download Si su valor se encuentra en true se descargará un archivo .json al entrar a la dirección web. Predeterminado: false

Ejemplos de uso

Si quisieramos obtener el Feed de noticias de la página oficial de InfoSmart en Facebook cuya dirección es: http://www.facebook.com/feeds/page.php?id=198535463518651&format=rss20 tendríamos que construir la dirección web:

%PATH%/to/xml_json?url=
¡Recuerde! Siempre puede usar la función urlencode() de PHP para codificar una dirección web de la forma correcta.

Esta dirección nos retornará el código transformado del XML a JSON listo para usarse en cualquier lenguaje de programación.

Usándolo en PHP

Dentro de PHP existe una función llamada json_decode que podemos usar para convertir una cadena de JSON a Array para poder manipular su información de forma sencilla.
