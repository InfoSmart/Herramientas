Tomar un pantallazo a la web

Funcionamiento

Esta herramienta te permite obtener la imagen de una simulación por navegador web o también llamado pantallazo de un sitio web.

La ventaja de esta herramienta es que puedes obtener de forma sencilla y rápida la imagen/pantallazo actualizada de cualquier sitio sin modulos pesados ni edición de código en tu servidor.

Ideas de uso

Esta herramienta puede ser ideal para mostrar una previa de cualquier sitio web tuyo o que haya sido registrado dentro de tu aplicación (por ejemplo: blogs o sitios patrocinados)

Parámetros

Dirección web base a solicitar:

%PATH%/to/screenshot

A partir de la dirección web (mostrada justo arriba) puedes agregar los siguientes parámetros:

  • url Dirección web del sitio a obtener su pantallazo. (Obligatorio)
  • width Ancho del pantallazo. (La altura se calculará automáticamente) Predeterminado: 300
    • Bug. Cambiar este parámetro puede ocasionar un error en ciertos sitios o sencillamente no hacer nada.
  • persistent Si su valor se encuentra en true se esperará hasta que el pantallazo se encuentre listo, de otra manera se mostrará una imagen de "Cargando...". Predeterminado: true
  • download Si su valor se encuentra en true el pantallazo se descargará al entrar a la dirección web. Predeterminado: false

Ejemplos de uso

Si quisieramos obtener el pantallazo de Facebook tendríamos que construir la dirección web:

%PATH%/to/screenshot?url=http://www.facebook.com/
¡Recuerde! Siempre puede usar la función urlencode() de PHP para codificar una dirección web de la forma correcta.

Esta dirección nos retornará el pantallazo con formato PNG del sitio "Facebook", nosotros podemos usarla en nuestro sitio de distintas formas, una de ellas es por una simple etiqueta img
