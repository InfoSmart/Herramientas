Obtener el Favicon de un sitio web

Funcionamiento

Esta herramienta te permite obtener el icono de identificación de un sitio web o también llamado favicon haciendo una simple petición al servicio almacenado localmente.

La ventaja de esta herramienta es que puedes obtener de forma sencilla y rápida el icono de cualquier sitio sin modulos pesados ni edición de código en tu servidor.

Ideas de uso

Esta herramienta puede ser perfecta para mostrar los iconos de tus sitios web en un tipo de portafolio o para usarlo en el código CSS de tu aplicación para mostrar los iconos de las redes sociales o sitios web populares.

InfoSmart usa esta tecnología en su página de productos.

Parámetros

Dirección web base a solicitar:

%PATH%/to/favicon

A partir de la dirección web (mostrada justo arriba) puedes agregar los siguientes parámetros:

  • url Dirección web del sitio a obtener su icono. (Obligatorio)
  • download Si su valor se encuentra en true el icono se descargará al entrar a la dirección web. Predeterminado: false
  • encode Si se ajusta su valor puede regresar el icono codificado en Base64 (base64) o MD5 (md5)

Ejemplos de uso

Si quisieramos obtener el favicon de Facebook tendríamos que construir la dirección web:

%PATH%/to/favicon?url=http://www.facebook.com/
¡Recuerde! Siempre puede usar la función urlencode() de PHP para codificar una dirección web de la forma correcta.
